Info

Blackjaq is a browser-based game using modified rules for the casino game Blackjack, or 21. The goal of the game is to have the hand whose value comes closest to 21 without going over, or equals 21.

It was made with HTML, CSS, JavaScript, jQuery, and ♥.

It's called Blackjaq because my name is Jaq, and I like puns.

close

Rules

  • The user must place an initial bet, and may continue to bet throughout the round.
  • Aces have a value of 1 at all times.
  • The user loses if they lose all their money.
  • The user wins if they reach $300.
